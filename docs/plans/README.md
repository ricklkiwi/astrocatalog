# Plans

This folder is for active implementation plans only.

A plan is active while its task is in planning, implementation, or review. Once the reviewer returns PASS, the orchestrator must move the completed task documents to:

```text
docs/archive/tasks/<slug>/
  plan.md
  spec.md
```

Do not leave completed task plans here for reference. Use the archive index instead:

- [Archived task documents](../archive/tasks/README.md)

- [PRD](../../planning/PRD-AstroTracker-v1.md)
- [Development plan](../../planning/development-plan.md)
- [Task breakdown](../../planning/task-breakdown.md)
